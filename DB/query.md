# Query작성 및 실행

1. Query 실행 할 Database 선택
2. Query Editor 선택
3. Query문 입력

```sql
SELECT * FROM [table_references];
```

> `*`은 모든 테이블에서 모든 열을 선택하기 위한 간단한 방법으로 사용 가능

4. Query 실행

   SQL 쿼리 도구 모음에서 왼쪽 3번째 아이콘 클릭
   Keyboard Shortcut : command+enter
