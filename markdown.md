# 마크다운 문법 정리

## Heading

### Markdown Syntax

```
# H1
## H2
### H3
```

## Bold

### Markdown Syntax

```
**bold text**
```

### Result

**bold**

## Italic

### Markdown Syntax

```
*italicized text*
```

### Result

_italic_

## Blockquote

### Markdown Syntax

```
> blockquote
```

### Result

> blockquote

## Ordered List

### Markdown Syntax

```
1. First item
2. Second item
3. Third item
```

### Result

1. First item
2. Second item
3. Third item

## Unordered List

### Markdown Syntax

```
- First item
- Second item
- Third item
```

### Result

- First item
- Second item
- Third item

## Code

### Markdown Syntax

```
`code`
```

### Result

`code`

## Horizontal Rule

### Markdown Syntax

```
---
```

### Result

---

## Link

### Markdown Syntax

```
[title](https://www.example.com)
```

### Result

[Google](https://www.google.com)

## Image

### Markdown Syntax

```
![alt text](image.jpg)
```

### Result

![Fox](1.jpeg)
